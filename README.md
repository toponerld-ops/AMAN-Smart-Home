## System Overview

**Module 1 - Smart Plant Care:**
- Soil moisture monitoring
- Auto-watering system
- Temperature & humidity tracking

**Module 2 - Room Security:**
- Motion detection
- Entry monitoring  
- Alert system

**Module 3 - Climate Monitor:**
- Multi-location temperature tracking
- Humidity monitoring
- Smart fan control

All modules communicate wirelessly to AMAN-OS central hub via Bluetooth.
